# Product Image Guide

## Where To Drop New Images
- Put all new raw images in: `assets/img/products/_incoming/`

## Naming Rules (important)
- Use lowercase only
- Use `-` instead of spaces
- Keep names short and descriptive
- Good examples:
  - `audio-earbuds-01.jpg`
  - `smartwatch-ultra-02.jpg`
  - `flash-sale-powerbank-01.jpg`

## Final Folders
- Audio section: `assets/img/products/audio/`
- Smartwatch section: `assets/img/products/smartwatch/`
- Flash sale visuals: `assets/img/products/flash-sale/`
- Empty cart/wishlist: `assets/img/products/empty-state/`
- Hero slides: `assets/img/products/hero/`

## Recommended File Formats
- Product photos: `.jpg` or `.webp`
- Transparent art/icons: `.png` or `.svg`
- Use `.webp` when possible for smaller size.

## Recommended Sizes
- Product cards: `800x800` (square)
- Hero / split art: `1600x900` (landscape)
- Empty-state illustrations: around `900x900`

## Quick Workflow
1. Add files to `_incoming`.
2. Rename them using the naming rules.
3. Move them to the correct folder.
4. Tell Codex the file names, and I will wire them into the site sections.
